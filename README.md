## 🎯 Getting Started

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/chillypriya-creator/hyperautomation.git
   cd hyperautomation
   ```

2. Open in your browser:
   ```bash
   open index.html
   # or
   python -m http.server 8000
   ```

3. Visit `http://localhost:8000`

### Deploy to GitHub Pages

1. Enable GitHub Pages in repository settings
2. Select `main` branch as the source
3. Your site will be available at: `https://chillypriya-creator.github.io/hyperautomation`

## 🎨 Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #8b5cf6;      /* Purple */
    --secondary-color: #ec4899;    /* Pink */
    --dark-bg: #0f172a;            /* Dark Blue */
    /* ... more colors ... */
}
```

### Content
Edit `index.html` to:
- Change company/project name
- Update feature descriptions
- Add your contact information
- Modify section content

### Styling
Edit `styles.css` to:
- Change fonts and typography
- Adjust spacing and layout
- Modify animations and transitions
- Customize color scheme
